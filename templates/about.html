Tentang Proyek Anisa

Ringkasan

Anisa adalah aplikasi web untuk mengklasifikasikan jenis jamur dari foto, dibangun sebagai proyek tugas kuliah. Sistem memanfaatkan transfer learning dari arsitektur Xception yang telah dilatih sebelumnya (pretrained) pada dataset ImageNet, kemudian dilakukan fine-tuning ringan pada dataset 12 spesies jamur.

Dataset

Dataset yang digunakan adalah 12 Mushroom Species Dataset dari Kaggle, diunduh secara otomatis menggunakan pustaka kagglehub. Untuk mempercepat proses pelatihan pada lingkup tugas kuliah, digunakan subset sekitar 150 gambar per kelas (≈ 1.000–2.000 gambar total dari 12 kelas), dibagi menjadi data latih (70%), validasi (15%), dan uji (15%).

Arsitektur Model

  1. Base model: Xception, include_top=False, bobot ImageNet, dibekukan pada tahap awal.
  2. Head classifier: Global Average Pooling → Dense(256, ReLU) → Dropout(0.3) → Dense(12, Softmax).
  3. Tahap 1: melatih head classifier saja (base dibekukan) selama beberapa epoch.
  4. Tahap 2: fine-tuning ringan — membuka 30 layer teratas Xception dengan learning rate sangat kecil.
  5. Regularisasi: EarlyStopping (memantau val_loss) & ModelCheckpoint (menyimpan model dengan val_accuracy terbaik).

Evaluasi

Model dievaluasi pada data uji yang belum pernah dilihat selama pelatihan. Metrik yang dilaporkan meliputi accuracy, classification report (precision, recall, F1-score per kelas), serta confusion matrix yang divisualisasikan sebagai heatmap. Semua artefak evaluasi tersimpan di folder model/ setelah train.py dijalankan.

Teknologi

Python Flask TensorFlow / Keras Xception (ImageNet) Bootstrap 5 kagglehub scikit-learn
